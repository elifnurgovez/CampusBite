# CampusBite 🌿🥪

## ## Problem
Üniversite kampüslerinde gıda israfı ve sürdürülebilirlik bilinci eksikliği yaşanmaktadır. Öğrenciler, artan malzemeleri nasıl değerlendireceklerini bilmemekte ve tükettikleri gıdaların çevresel etkilerini (su ayak izi, CO2 emisyonu) takip edememektedir.

## ## Çözüm
CampusBite, üniversite öğrencileri için geliştirilmiş yapay zeka tabanlı bir **Eco-Assistant** uygulamasıdır. Kullanıcılar, ellerindeki malzemelerle atıksız tarifler alabilir ve gıdaların çevresel etkilerini anlık olarak analiz edebilirler. Google Gemini API ve Supabase entegrasyonu sayesinde her öğrencinin kampüs yaşamını daha yeşil bir hale getirmesini sağlar.

## ## Kullanılan Teknolojiler
- **Framework:** Next.js (App Router)
- **Yapay Zeka:** Google Gemini API (2.0 Flash)
- **Veritabanı:** Supabase (PostgreSQL)
- **Stil:** Tailwind CSS
- **Dağıtım:** Vercel

## ## Nasıl Çalıştırılır?
1. Projeyi klonlayın: `git clone https://github.com/kullanici-adin/CampusBite.git`
2. Bağımlılıkları yükleyin: `npm install`
3. `.env.local` dosyasına API anahtarlarınızı ekleyin.
4. Uygulamayı başlatın: `npm run dev`
