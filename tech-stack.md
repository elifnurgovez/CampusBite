# CampusBite - Teknoloji Yığını (Tech Stack)

## Frontend (Arayüz)
- **Framework:** Next.js (App Router) - *Hızlı ve modern olduğu için.*
- **Styling:** Tailwind CSS - *Tasarımı tek satır kodla halletmek için.*
- **Components:** Lucide-react (İkonlar için).

## Backend & Database (Arka Plan)
- **Platform:** Supabase - *Veritabanı (PostgreSQL) ve Auth işlemlerini tek yerden çözdüğü için.*
- **Language:** JavaScript/TypeScript.

## AI (Yapay Zeka)
- **Model:** Google Gemini 2.0 Flash - *En güncel, hızlı ve ücretsiz API kotası olduğu için.*
- **Entegrasyon:** Google AI SDK.

## Hosting (Yayınlama)
- **Platform:** Vercel - *Next.js projesini 30 saniyede yayına aldığı için.*
